# The Tingey Family Sunriver Escape

A mobile-first GitHub Pages vacation site for Sunriver, Oregon — August 5–10, 2026.

## How to use
1. Upload `index.html`, `styles.css`, and `script.js` to a GitHub repository.
2. In GitHub, go to Settings → Pages.
3. Choose your main branch and root folder.
4. Save and open the Pages URL after GitHub publishes it.

## Easy edits
- Change trip dates in `index.html` and the countdown target in `script.js`.
- Replace photo URLs in each `<img src="...">` line with your own images from an `/images` folder.
- Add your lodging address to the map buttons if you want one-tap directions from the house.
